# 🤖 Silveira Atendimento Inteligente

🚀 Landing page desenvolvida para apresentar uma solução de atendimento inteligente utilizando Inteligência Artificial.

## 🛠️ Tecnologias utilizadas

- ⚛️ React
- ⚡ Vite
- 🟨 JavaScript
- 🌐 Hostinger Horizons (Vibe Coding)
- 🤖 Integração com agente de IA

## 🎯 Objetivo do projeto

Demonstrar a criação de um produto digital completo, unindo:

- 🎨 Interface moderna e responsiva
- 🧠 Experiência do usuário (UX)
- ⚙️ Automação inteligente de atendimento
- 💬 Solução baseada em inteligência artificial para empresas

## 📌 Sobre o projeto

A Silveira Atendimento Inteligente é uma solução criada para demonstrar como agentes de IA podem auxiliar empresas a melhorar seu atendimento ao cliente, oferecendo respostas rápidas, personalizadas e disponíveis 24 horas por dia.
